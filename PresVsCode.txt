Qu'est-ce que Visual Code (Vscode) ? qsqjnbjknde

- Un éditeur de code "cross-platform" (plusieurs plates-formes), open source sous licence MIT (MIT, car elle provient de la Massachusetts Institute of Technology), gratuit présenter en avril 2015 à la conf des dév BUILD 
- Code source fournit sur GitHub, cependant, l'exécutable est privé et proposé par Microsoft. 
- Il supporte une dizaine de langages dont : JavaScript, HTML, CSS, par exemple, mais beaucoup d'autres, grâce au marché d'extensions offertes par la communauté travaillant sur le projet VScode sur GitHub. 
- UN IDE (Integrated development environment) qui se place en tête des projets avec un grand nombre de contributeurs sur la plateforme GitHub 
- Un éditeur de code écrit en TypeScript et JavaScript (alors que par exemple Sublime Text est écrit en C++ et Python) 
- Pricipal Point Faible : Inadapté à la manipulation de gros fichiers de code 
- Principal Point Fort : Architecture à base d'extension 
- IDE très léger, pouvant être étendue selon les besoins de son utilisateur 
- Un moteur de gestion très puissant, grâce notamment à un moteur d'auto-complétion de code (IntelliSense) 
- Une interface qui communique par défaut avec Git 
- Lors d'une transition d'un utilisateur d'un éditeur de code différent pour VsCode, celui-ci peut importer si besoin les raccourcis clavier de ses autres éditeurs 
- VsCode est dans un processus de DEVELOPPEMENT CONTINU (2maj/mois) 
- // SI Microsoft s'est lancé dans le projet, c'est dans le but de racommunauté GitHub.

Conclusion : VsCode est un éditeur de code fonctionnel, supportant pas mal de langages et de fonctionnalités faisant gagnées en temps et en productivité et étant personnalisable en fonction des besoins de son utilisateur, grâce à des extensions créées par la communauté GitHub.
